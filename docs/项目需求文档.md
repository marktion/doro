# 语音交互系统 - 项目需求文档

## 1. 项目概述

### 1.1 项目名称
Doro - Claude Code 语音交互界面

### 1.2 项目目标
为 Claude Code 提供语音交互能力，将命令行的文字交互改为语音交互：
- **语音输入 → MiMo ASR识别 → Agent SDK → Claude Code输出 → MiMo TTS合成 → 语音播放**

### 1.3 核心价值
- 无需打字，语音即可与Claude Code对话
- 解放双手，适合多任务场景
- 保持Claude Code的完整能力（代码、文件、工具等）
- 生成式UI：根据输出类型动态渲染组件

### 1.4 项目本质
本项目是 Claude Code 的**语音交互前端**，类似于：
- 把命令行的"文字输入"改成"语音输入"
- 把命令行的"文字输出"改成"语音播放" + "动态UI"
- Claude Code 本身作为"大脑"负责思考和执行

### 2.3 API接口详情

#### ASR（语音识别）
- **端点**：`POST https://api.xiaomimimo.com/v1/chat/completions`
- **模型**：`mimo-v2.5-asr`
- **认证**：`api-key: {MIMO_API_KEY}`
- **音频格式**：WAV、MP3
- **语言参数**：`asr_options.language`（支持 `zh` 中文、`en` 英文等）
- **音频传输**：Base64编码，通过 `input_audio.data` 字段传入
- **音频参数**：采样率16000Hz，单声道，16bit
- **响应格式**：
  ```json
  {
    "choices": [{
      "message": {
        "content": "识别结果文字",
        "role": "assistant"
      }
    }],
    "usage": { "prompt_tokens": 100, "completion_tokens": 50 }
  }
  ```
- **流式响应**：不支持流式返回，每次返回完整识别结果
- **中间结果**：无中间结果，录音结束后一次性返回最终结果

#### TTS（语音合成）
- **端点**：`POST https://api.xiaomimimo.com/v1/audio/speech`
- **模型**：`mimo-tts`
- **认证**：`api-key: {MIMO_API_KEY}`
- **套餐**：全档位限时免费
- **响应格式**：MP3音频流

#### Agent SDK（Claude Code集成）
- **包名**：`@anthropic-ai/claude-agent-sdk`
- **版本**：0.3.170（2026-06-09）
- **Node.js要求**：>= 18.0.0
- **模块类型**：ESM

### 2.4 平台优势
1. **Agent SDK**：结构化消息，无需解析ANSI码
2. **完整会话管理**：listSessions, resume, deleteSession等API
3. **生成式UI支持**：工具调用作为结构化数据，可动态渲染
4. **TTS限时免费**：成本优惠
5. **中文优化**：原生支持中文，识别和合成效果好

### 3.2 数据流

```
用户说话
    │
    ▼
[MediaRecorder] ──→ 音频数据 (WebM/Opus)
    │                  格式: audio/webm;codecs=opus
    │                  采样率: 16000Hz
    │                  声道: 单声道
    ▼
[WebSocket: 二进制帧] ──→ 发送到后端
    │                       完整音频帧（非分片）
    ▼
[后端接收] ──→ 合并音频帧（如有多个）
    │           转换为Base64
    │           添加data URL前缀
    ▼
[MiMo ASR API] ──→ mimo-v2.5-asr 模型
    │                  (POST /v1/chat/completions)
    │                  音频格式: data:audio/webm;base64,{BASE64_AUDIO}
    ▼
[识别结果] ──→ 文字 ("帮我创建一个React组件")
    │           一次性返回最终结果（无流式）
    ▼
[WebSocket: asr:result] ──→ 前端显示识别文字
    │
    ▼
[Agent SDK query()] ──→ 结构化消息流
    │                      SDKMessage: stream_event, assistant, result
    ▼
[WebSocket: agent:message] ──→ 前端动态渲染
    │                           stream_event → 流式文本
    │                           assistant → 完整消息 + 工具调用
    │                           result → 最终结果
    ▼
[MiMo TTS API] ──→ mimo-tts 模型
    │                  (文字转语音)
    │                  句子级分块合成
    ▼
[WebSocket: 二进制音频帧] ──→ 前端播放音频
    │                           流式播放，低延迟
    ▼
[用户听到回复]
```

### 3.3 Agent SDK消息类型

| 消息类型 | 说明 | 前端处理 |
|---------|------|----------|
| `stream_event` | 流式token | 实时文本渲染 |
| `assistant` | 完整消息（文本+工具调用） | 渲染文本块 + 工具组件 |
| `result` | 最终结果（成功/失败） | 显示成本、状态 |
| `system` | 系统事件（init等） | 渲染系统状态 |
| `tool_progress` | 工具执行进度 | 显示进度条 |

**句子级TTS分块规则**：
```typescript
// 分句标点
const SENTENCE_DELIMITERS = /[。！？.!?]+/;

// 分句函数
function splitIntoSentences(text: string): string[] {
  const sentences: string[] = [];
  let current = '';
  
  for (const char of text) {
    current += char;
    if (SENTENCE_DELIMITERS.test(char)) {
      sentences.push(current.trim());
      current = '';
    }
  }
  
  if (current.trim()) {
    sentences.push(current.trim());
  }
  
  return sentences;
}

// 长文本截断阈值
const MAX_SENTENCE_LENGTH = 500; // 字符

function splitLongSentence(sentence: string): string[] {
  if (sentence.length <= MAX_SENTENCE_LENGTH) {
    return [sentence];
  }
  
  // 按逗号、分号等次级标点拆分
  const subDelimiters = /[，、；,;]+/;
  const parts: string[] = [];
  let current = '';
  
  for (const char of sentence) {
    current += char;
    if (subDelimiters.test(char) && current.length > 100) {
      parts.push(current.trim());
      current = '';
    }
  }
  
  if (current.trim()) {
    parts.push(current.trim());
  }
  
  return parts;
}

// 播放策略：串行播放，避免重叠
class TTSPlayer {
  private queue: string[] = [];
  private isPlaying = false;
  
  async addToQueue(text: string) {
    const sentences = splitIntoSentences(text);
    for (const sentence of sentences) {
      const parts = splitLongSentence(sentence);
      this.queue.push(...parts);
    }
    
    if (!this.isPlaying) {
      this.playNext();
    }
  }
  
  private async playNext() {
    if (this.queue.length === 0) {
      this.isPlaying = false;
      return;
    }
    
    this.isPlaying = true;
    const text = this.queue.shift()!;
    
    try {
      const audioData = await fetchTTS(text);
      await this.playAudio(audioData);
    } catch (error) {
      console.error('TTS播放失败:', error);
    }
    
    // 播放下一句
    this.playNext();
  }
  
  private playAudio(data: ArrayBuffer): Promise<void> {
    return new Promise((resolve) => {
      const audio = new Audio(URL.createObjectURL(new Blob([data])));
      audio.onended = resolve;
      audio.play();
    });
  }
}
```

```typescript
// 1. Agent SDK中断
class AgentController {
  private abortController: AbortController | null = null;
  
  async interrupt() {
    if (this.abortController) {
      this.abortController.abort();
      this.abortController = null;
    }
  }
  
  async query(prompt: string, options: any) {
    this.abortController = new AbortController();
    
    const result = query({
      prompt,
      options: {
        ...options,
        abortSignal: this.abortController.signal
      }
    });
    
    return result;
  }
}

// 2. TTS音频中断
class TTSInterrupter {
  private audioContext: AudioContext | null = null;
  private sourceNode: AudioBufferSourceNode | null = null;
  private bufferQueue: ArrayBuffer[] = [];
  
  interrupt() {
    // 停止当前播放
    if (this.sourceNode) {
      this.sourceNode.stop();
      this.sourceNode = null;
    }
    
    // 清空缓冲区
    this.bufferQueue = [];
    
    // 暂停AudioContext
    if (this.audioContext) {
      this.audioContext.suspend();
    }
  }
  
  resume() {
    if (this.audioContext) {
      this.audioContext.resume();
    }
  }
}

// 3. WebSocket音频帧清空
class WebSocketAudioManager {
  private ws: WebSocket;
  
  interrupt() {
    // 发送中断信号
    this.ws.send(JSON.stringify({ type: 'query:interrupt' }));
    
    // 清空待发送队列
    this.messageQueue = [];
    
    // 标记当前任务为已中断
    this.isInterrupted = true;
  }
  
  onMessage(data: any) {
    if (this.isInterrupted) {
      // 忽略中断后的消息
      return;
    }
    
    // 处理消息
  }
}
```

#### WebSocket健壮性机制

**心跳保活**：
```typescript
// 客户端：每30秒发送ping
setInterval(() => {
  if (ws.readyState === WebSocket.OPEN) {
    ws.send(JSON.stringify({ type: 'ping', timestamp: Date.now() }));
  }
}, 30000);

// 服务器：收到ping立即返回pong
ws.on('message', (data) => {
  const msg = JSON.parse(data);
  if (msg.type === 'ping') {
    ws.send(JSON.stringify({ type: 'pong', timestamp: msg.timestamp }));
  }
});
```

**自动重连（指数退避）**：
```typescript
// 客户端重连逻辑
let reconnectAttempts = 0;
const maxReconnectAttempts = 10;
const baseDelay = 1000; // 1秒

function reconnect() {
  if (reconnectAttempts >= maxReconnectAttempts) {
    console.error('重连次数超限');
    return;
  }
  
  const delay = baseDelay * Math.pow(2, reconnectAttempts);
  setTimeout(() => {
    ws = new WebSocket(wsUrl);
    reconnectAttempts++;
  }, delay);
}
```

**断连恢复**：
```typescript
// 断连时暂停任务
ws.onclose = () => {
  // 暂停当前录音
  if (mediaRecorder && mediaRecorder.state === 'recording') {
    mediaRecorder.pause();
  }
  
  // 暂停TTS播放
  if (audioContext && audioContext.state === 'running') {
    audioContext.suspend();
  }
  
  // 记录未完成的任务
  pendingTasks.push({
    type: 'query',
    data: currentQueryData,
    timestamp: Date.now()
  });
  
  reconnect();
};

// 重连后恢复任务
ws.onopen = () => {
  // 恢复录音
  if (mediaRecorder && mediaRecorder.state === 'paused') {
    mediaRecorder.resume();
  }
  
  // 恢复TTS播放
  if (audioContext && audioContext.state === 'suspended') {
    audioContext.resume();
  }
  
  // 重发未完成的任务
  pendingTasks.forEach(task => {
    ws.send(JSON.stringify(task));
  });
  pendingTasks = [];
};
```

**消息队列（网络抖动）**：
```typescript
// 客户端消息队列
const messageQueue: string[] = [];
const isOnline = navigator.onLine;

function sendMessage(data: any) {
  if (isOnline && ws.readyState === WebSocket.OPEN) {
    ws.send(JSON.stringify(data));
  } else {
    messageQueue.push(JSON.stringify(data));
  }
}

// 网络恢复时清空队列
window.addEventListener('online', () => {
  while (messageQueue.length > 0) {
    const msg = messageQueue.shift();
    if (ws.readyState === WebSocket.OPEN) {
      ws.send(msg);
    }
  }
});
```

```typescript
const RECOMMENDED_MIME_TYPE = 'audio/webm;codecs=opus';
const FALLBACK_MIME_TYPE = 'audio/webm';
const SAMPLE_RATE = 16000; // 16kHz
const CHANNELS = 1; // 单声道

// 检测浏览器支持
function getSupportedMimeType(): string {
  const types = [
    'audio/webm;codecs=opus',
    'audio/webm',
    'audio/mp4',
    'audio/wav'
  ];
  
  for (const type of types) {
    if (MediaRecorder.isTypeSupported(type)) {
      return type;
    }
  }
  
  throw new Error('浏览器不支持音频录制');
}

// 创建MediaRecorder配置
function createMediaRecorderOptions(): MediaRecorderOptions {
  const mimeType = getSupportedMimeType();
  
  return {
    mimeType,
    audioBitsPerSecond: 128000, // 128kbps
  };
}

// 音频参数验证
function validateAudioParams(audioBuffer: AudioBuffer): boolean {
  // 检查采样率
  if (audioBuffer.sampleRate !== SAMPLE_RATE) {
    console.warn(`采样率不匹配: ${audioBuffer.sampleRate} != ${SAMPLE_RATE}`);
    return false;
  }
  
  // 检查声道数
  if (audioBuffer.numberOfChannels !== CHANNELS) {
    console.warn(`声道数不匹配: ${audioBuffer.numberOfChannels} != ${CHANNELS}`);
    return false;
  }
  
  return true;
}
```

### 15.2 P0问题：音频链路转换规则

**问题**：前端传WebM/WAV二进制流，MiMo ASR要求Base64 + data URL格式。

**解决方案**：
```typescript
// 前端：录制音频
async function startRecording(): Promise<MediaRecorder> {
  const stream = await navigator.mediaDevices.getUserMedia({
    audio: {
      sampleRate: 16000,
      channelCount: 1,
      echoCancellation: true,
      noiseSuppression: true
    }
  });
  
  const mimeType = getSupportedMimeType();
  const recorder = new MediaRecorder(stream, { mimeType });
  
  const chunks: Blob[] = [];
  
  recorder.ondataavailable = (e) => {
    if (e.data.size > 0) {
      chunks.push(e.data);
    }
  };
  
  recorder.onstop = async () => {
    // 合并音频分片
    const audioBlob = new Blob(chunks, { type: mimeType });
    
    // 转换为ArrayBuffer
    const arrayBuffer = await audioBlob.arrayBuffer();
    
    // 发送到后端（二进制帧）
    ws.send(arrayBuffer);
  };
  
  recorder.start(100); // 每100ms触发一次dataavailable
  return recorder;
}

// 后端：接收并转换音频
async function processAudio(audioBuffer: Buffer): Promise<string> {
  // 转换为Base64
  const base64Audio = audioBuffer.toString('base64');
  
  // 添加data URL前缀
  const dataUrl = `data:audio/webm;base64,${base64Audio}`;
  
  return dataUrl;
}

// 后端：调用ASR API
async function callASR(audioDataUrl: string): Promise<string> {
  const response = await fetch('https://api.xiaomimimo.com/v1/chat/completions', {
    method: 'POST',
    headers: {
      'api-key': process.env.MIMO_API_KEY!,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      model: 'mimo-v2.5-asr',
      messages: [{
        role: 'user',
        content: [{
          type: 'input_audio',
          input_audio: {
            data: audioDataUrl
          }
        }]
      }],
      asr_options: {
        language: 'zh'
      }
    })
  });
  
  const data = await response.json();
  return data.choices[0].message.content;
}
```

### 15.3 P0问题：语音打断中断逻辑

```typescript
// 1. Agent SDK中断
class AgentController {
  private abortController: AbortController | null = null;
  
  async query(prompt: string, options: any) {
    this.abortController = new AbortController();
    
    const result = query({
      prompt,
      options: {
        ...options,
        abortSignal: this.abortController.signal
      }
    });
    
    return result;
  }
  
  interrupt() {
    if (this.abortController) {
      this.abortController.abort();
      this.abortController = null;
    }
  }
}

// 2. TTS音频中断
class TTSPlayer {
  private audioContext: AudioContext | null = null;
  private sourceNode: AudioBufferSourceNode | null = null;
  private bufferQueue: ArrayBuffer[] = [];
  
  async playAudio(data: ArrayBuffer) {
    if (!this.audioContext) {
      this.audioContext = new AudioContext();
    }
    
    const audioBuffer = await this.audioContext.decodeAudioData(data);
    this.sourceNode = this.audioContext.createBufferSource();
    this.sourceNode.buffer = audioBuffer;
    this.sourceNode.connect(this.audioContext.destination);
    
    return new Promise<void>((resolve) => {
      this.sourceNode!.onended = resolve;
      this.sourceNode!.start();
    });
  }
  
  interrupt() {
    // 停止当前播放
    if (this.sourceNode) {
      this.sourceNode.stop();
      this.sourceNode = null;
    }
    
    // 清空缓冲区
    this.bufferQueue = [];
    
    // 暂停AudioContext
    if (this.audioContext) {
      this.audioContext.suspend();
    }
  }
  
  resume() {
    if (this.audioContext) {
      this.audioContext.resume();
    }
  }
}

// 3. WebSocket音频帧清空
class WebSocketManager {
```

```typescript
  }

// 前端：边录边显实时文字的实现方案
class RealtimeASR {
  private recordingTimer: number | null = null;
  private currentText = '';
  
  startRecording() {
    // 每2秒发送一次音频进行识别（模拟流式）
    this.recordingTimer = window.setInterval(async () => {
      if (this.mediaRecorder && this.mediaRecorder.state === 'recording') {
        // 停止当前录制
        this.mediaRecorder.stop();
        
        // 获取音频数据
        const audioData = await this.getAudioData();
        
        // 发送到后端进行识别
        const partialText = await this.recognizePartial(audioData);
        
        // 更新显示（显示为"临时"状态）
        this.updateDisplay(partialText, false);
        
        // 重新开始录制
        this.mediaRecorder.start(100);
      }
    }, 2000);
  }
  
  stopRecording() {
    if (this.recordingTimer) {
      clearInterval(this.recordingTimer);
      this.recordingTimer = null;
    }
    
    // 最后一次识别（最终结果）
    this.recognizeFinal().then((finalText) => {
      this.updateDisplay(finalText, true);
    });
  }
  
  private updateDisplay(text: string, isFinal: boolean) {
    // 前端显示逻辑
    // isFinal=false: 显示为灰色（临时结果）
    // isFinal=true: 显示为黑色（最终结果）
    this.currentText = text;
    this.renderText(text, isFinal);
  }
}
```
